# 🌍 AI Travel Planner

An AI-powered travel planner built using Python, Streamlit, and the OpenAI API. It generates highly customized, professional, day-by-day itineraries tailored to your destination, trip duration, travel style, and budget.

## ✨ Features

- 🗺️ **Personalized Itineraries:** Generates detailed morning, afternoon, and evening schedules.
- 🎨 **Tailored Experiences:** Choose between multiple travel styles (Adventure, Cultural, Relaxed, etc.) and budget configurations (Economy, Moderate, Luxury).
- 🍽️ **Local Dining Options:** Curates breakfast, lunch, and dinner recommendations.
- 💰 **Smart Logistics:** Provides estimated daily budgets, local transit advice, and destination-specific packing tips.
- 📥 **Export Functionality:** Easily download your generated itinerary as a text file.

## 🛠️ Technologies Used

- **Python**
- **Streamlit** (Frontend Interface)
- **OpenAI API** (Itinerary Engine)
- **python-dotenv** (Environment Variable Management)
